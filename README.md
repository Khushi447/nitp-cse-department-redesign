# NIT Patna CSE Department Website (Redesign)

This is a modern redesign of the Computer Science & Engineering Department website of NIT Patna.

## 🚀 Tech Stack
- React (Vite)
- Tailwind CSS
- React Router
- Lucide Icons

## ✨ Features
- Responsive modern UI
- Interactive stats section
- Research & People pages
- Clean footer and navigation
